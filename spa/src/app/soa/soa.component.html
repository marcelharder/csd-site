Surgical Outcomes Application

Motivation:
This application is designed to measure the surgical outcomes for the individual cardiac surgeon.
The surgeon can monitor the outcomes in real time and these data are private.
Because this application is surgeon-centric the application has no dependencies on the
hospital where the surgeon works. View it as a surgical logbook with advanced capabilities.
